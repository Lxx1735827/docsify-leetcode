## 创建一个文件夹

## 在文件夹中克隆项目地址
> git clone https://github.com/Lxx1735827/docsify-leetcode.git

## 创建自己的分支并切换
> git branch lxx(这里换成自己的名字)
> git checkout lxx(这里换成自己的名字)

## 在本地编辑之后执行以下命令
> git add .
> git commit -m "你的描述"
> git push -u origin 你的分支

## 注意
> 每次编写之前，先拉取master分支的内容
> git pull origin master
